## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start
   ```

3. package
   ```bash
   eas secret:push --scope project --env-file .env
   eas build -p android
   eas build -p android --local
   eas build -p ios
   ```
